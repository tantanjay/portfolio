Modern Image Classification Pipeline (Personal Project)

A personal project to dive into the world of artificial intelligence by building a complete, end-to-end pipeline for training image classification models. The system is designed to take any custom image dataset, automatically train a deep learning model to recognize the contents, and produce a detailed report on its performance.

  • Roles: Solo Developer / AI Developer
  • Project Goal: To self-learn and implement a modern, reusable deep learning training pipeline from scratch.
  • Languages & Frameworks:
    • Python: The core language for the entire project.
    • TensorFlow & Keras: Used to build, train, and evaluate the neural network (EfficientNetV2).
  • Technologies: Pandas, Scikit-learn, Matplotlib, Seaborn, PyYAML
  • Learning Journey:
    • Data Pipeline: Built a robust data loader that automatically reads, splits, and preprocesses thousands of images. Implemented advanced data augmentation techniques like RandAugment to prevent overfitting and improve model accuracy.
    • Model Architecture: Engineered a flexible model builder that uses transfer learning from the powerful EfficientNetV2 model and attaches a custom classification head.
    • Advanced Training: Implemented sophisticated training strategies, including "Gradual Unfreezing" and learning rate schedulers like "Cosine Decay," to maximize model performance.
    • Comprehensive Evaluation: Created an automated evaluation module that generates a suite of visualizations, including a detailed 8-graph training dashboard, confusion matrices, and per-class performance reports to thoroughly analyze the model's strengths and weaknesses.
